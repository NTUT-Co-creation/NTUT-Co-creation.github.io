1使い方

各事例ページは styles.css を1行読み込むだけで体裁が揃います。CSS本体は編集せず、HTML側のクラスを組み合わせて使います。

<link rel="stylesheet" href="styles.css">

新しい事例ページは jma.html を複製し、中身（連携先名・本文・写真）を差し替えるのが最短です。基本の骨組みは次の通り。

<div class="sheet">
  <header class="case-header">  <!-- banner --> … 学科・連携先名・期間
  <main class="case-main">     <!-- main -->
    <section class="hero"> … 見出しコピー＋主写真 </section>
    <div class="case-body"> … 本文・情報パネル・カード・成果 </div>
  </main>
  <footer class="case-footer"> <!-- contentinfo --> … 出典・コピーライト
</div>

この header（banner）→ main → footer（contentinfo） の3ランドマーク構成により、スクリーンリーダー利用者がキー操作でページの主要部へ素早く移動できます。見出しは h1（連携先名）→ h2（各セクション）→ h3（カード見出し）の順を崩さないでください。

2配色トークン

色はすべて CSS変数（デザイントークン）で管理しています。直接カラーコードを書かず、下の変数名を使ってください。

--course-tint#f6e2e9 面の淡ピンク
--course-soft#fbf1f4 さらに淡い差し色
--course-deep#c76b87 ラベル・罫線
--course-deep2#a64f6c 見出しの強調
--ink#2b2b2b 本文
--ink-soft#5d5d5d 補足・キャプション
--paper#ffffff 紙面の白

3コース色の変え方

コース（学科・領域）ごとに色を分けます。<body> にクラスを付けるだけで切替わり、CSSの編集は不要です。配色はパンフレット「産業技術学部」の各コース色を元にし、すべて WCAG AA（コントラスト比4.5以上）を満たします。全学部・全領域・横断の事例はクラス無し（＝既定のグレー）でOK。

情報科学コース（青）
theme-info / tint #d7edf4 / deep #017ca2
先端機械工学コース（緑）
theme-mech / tint #e0eedd / deep #45733a
建築学コース（赤）
theme-arch / tint #f3d8da / deep #842a2f
クリエイティブデザイン学（橙）
theme-design / tint #f4e3d7 / deep #a54709
支援技術学コース（青緑）
theme-support / tint #d7f3f4 / deep #028283
全学部・全領域・横断（グレー・既定）
theme-gray / tint #e6e2df / deep #54504b

使い方

<body> にコースのクラスを付けるだけ。全学部・全領域・横断はクラス無し（既定グレー）。

<body class="theme-info">     <!-- 情報科学コース（青）   -->
<body class="theme-mech">     <!-- 先端機械工学コース（緑） -->
<body class="theme-arch">     <!-- 建築学コース（赤）     -->
<body class="theme-design">   <!-- クリエイティブデザイン学（橙）-->
<body class="theme-support">  <!-- 支援技術学コース（青緑） -->
<body>  または class="theme-gray"  <!-- 全学部・全領域・横断（グレー）-->

グレー（全学部）の扱い方
淡いグレーをアクセントにも使うと「無効・非活性」に見えて地味です。そこで アクセント側（deep / deep2）だけは黒に近いチャコールに。背景は淡グレーのまま、見出し・罫線・成果バッジをチャコールにすると引き締まり、白文字も読めます（白文字 on #54504b ＝ 8.0:1）。少し暖色寄り（greige）の #e6e2df で紙の質感ともなじみます。

パンフレット色 → 調整シェードの考え方
パンフレットの各コース色（青 #019aca／緑 #75b266／赤 #cb5b61／橙 #f47b2a／青緑 #02abad）は鮮やかなため、そのままでは白文字が読めません。背景 tint/soft はその色を淡くし、deep/deep2 は白文字・見出しが読める濃さに調整しています。学章色（DIC）はロゴ・トリコロール装飾用に --brand-* として別に保持し、トップページの3色帯などに使います。

⚠ コントラストの基準（色を新規に作るとき）
deep には白文字（成果バッジ・期間ラベル）が載ります。白文字とのコントラスト比が 4.5:1 以上 に。deep2（見出し文字）も tint・soft・白 の各背景に対して 4.5:1 以上を確保します。上の6テーマはすべてこの基準を満たしています。 WebAIM Contrast Checker で確認できます。

6ファイル構成

フォルダごと配布・公開できます。サーバーに置けばそのまま動きます。

web/
├─ styles.css　…… デザインシステム本体（共通・編集不要）
├─ style-guide.html　…… このページ（見本と使い方）
├─ jma.html　…… 気象庁の事例ページ（複製のお手本）
└─ images/　…… 各事例の写真

写真は images/ に入れ、HTMLからは images/ファイル名.jpg の相対パスで参照します。1枚目（主写真）と意見交換などの記録写真を .figure で配置してください。
