1使い方

各事例ページは styles.css を1行読み込むだけで体裁が揃います。CSS本体は編集せず、HTML側のクラスを組み合わせて使います。

<link rel="stylesheet" href="styles.css">

新しい事例ページは jma.html を複製し、中身（連携先名・本文・写真）を差し替えるのが最短です。基本の骨組みは次の通り。

<div class="sheet">
  <header class="case-header">  <!-- banner --> … 学科・連携先名・期間
  <main class="case-main">     <!-- main -->
    <section class="hero"> … 見出しコピー＋主写真 </section>
    <div class="case-body"> … 本文・情報パネル・カード・成果 </div>
  </main>
  <footer class="case-footer"> <!-- contentinfo --> … 出典・コピーライト
</div>

この header（banner）→ main → footer（contentinfo） の3ランドマーク構成により、スクリーンリーダー利用者がキー操作でページの主要部へ素早く移動できます。見出しは h1（連携先名）→ h2（各セクション）→ h3（カード見出し）の順を崩さないでください。

2配色トークン

色はすべて CSS変数（デザイントークン）で管理しています。直接カラーコードを書かず、下の変数名を使ってください。

--course-tint#f6e2e9 面の淡ピンク
--course-soft#fbf1f4 さらに淡い差し色
--course-deep#c76b87 ラベル・罫線
--course-deep2#a64f6c 見出しの強調
--ink#2b2b2b 本文
--ink-soft#5d5d5d 補足・キャプション
--paper#ffffff 紙面の白

3コース色の変え方

コース（学科）ごとに背景色を変えるには、<body> に変数を上書きするだけ。CSSの編集は不要で、同じコースの4事例は1色で統一します。パンフレット（2ページ目）に合わせた公式3テーマを用意しました。

ピンク（既定）
tint #f6e2e9 / deep #b04d6a
オレンジ
tint #f3b76f / deep #9a5612
グレー（チャコール）
tint #e6e2df / deep #54504b
青（学章 DIC138）
tint #d2ecf7 / deep #006d96
緑（学章 DIC572）
tint #d6f0e0 / deep #00773a
黄（学章 DIC125）
tint #ffdf7a / deep #8a6500

① ピンク（既定）

CSSの初期値。上書き不要でそのまま使えます。

<body>  <!-- 何も指定しなければピンク -->

② オレンジ

パンフレットのオレンジ #f3b76f をそのまま背景に。アクセントは濃い茶系で締めます。

<body style="--course-tint:#f3b76f; --course-soft:#fdf1e2;
             --course-deep:#9a5612; --course-deep2:#6e3d05;">

③ グレー（チャコール）

背景は暖かみのある淡グレー、アクセントは黒に近いチャコールで。

<body style="--course-tint:#e6e2df; --course-soft:#f3f0ee;
             --course-deep:#54504b; --course-deep2:#3c3935;">

④ 学章カラー（青・緑・黄）

筑波技術大学の学章色（青 DIC138 / 緑 DIC572 / 黄 DIC125）を元にしたテーマ。 クラス名 theme-blue / theme-green / theme-yellow で切り替えます。 

<body class="theme-blue">    <!-- 青 DIC138 -->
<body class="theme-green">   <!-- 緑 DIC572 -->
<body class="theme-yellow">  <!-- 黄 DIC125 -->

学章色そのものは --brand-blue / --brand-green / --brand-yellow
正確なブランド色（#00a4d6 / #009f41 / #ffd03a）はロゴ・図版・アクセント用に --brand-* として別に用意しています。ただし学章色は鮮やかな原色のため白文字を載せると読めません（白文字 on 黄＝1.5:1、青＝2.9:1）。そこで背景は学章色を淡くした tint に、見出しや白文字バッジは濃く調整した deep にしています。

黄色の扱い方（グレーと同じ考え方）
黄色は明るすぎて白文字が絶対に乗りません。そこで deep だけは黄を深めたダークアンバー #8a6500 にして、成果バッジや期間ラベルの白文字を確保しています（白文字 5.3:1）。背景の鮮やかな黄色はそのまま個性として活かせます。

⑤ 学章3色をまとめて使う（ブランドページ）

トップページや表紙など、学章の3色を1ページで前面に出したいときの組み合わせ方です。実例は index.html（ブランドトップ） を参照。

青ブロック
白文字 5.8:1
緑ブロック
白文字 5.7:1
黄ブロック
白文字 5.3:1

3色をまとめるコツは4つ：①白を基調（約6割）にして3色は“差し色”に留める／②並べる順番は学章どおり 青→緑→黄 で固定／③色面に文字を載せるときは調整シェード（deep）を使う（鮮やかな原色 --brand-* は線・帯・ドットなど文字が乗らない装飾だけに）／④黄の面の文字は暗色に。

再利用できる部品：.brand-rule（水平のトリコロール帯）、.brand-rule--diagonal（斜めのトリコロール帯＝パンフレットの斜めモチーフ）、.brand-dots（3色ドット）。装飾なので aria-hidden="true" を付けて読み上げから除外します。

グレーの扱い方（ポイント）
無彩色は、淡いグレーをアクセントにも使うと「灰色＝無効・非活性」に見え、地味で識別しづらくなります。そこで アクセント側（deep / deep2）だけは黒に近いチャコールにするのがコツです。背景は淡グレーのまま、見出し・罫線・成果バッジ・斜めマーカーをチャコールにすることで、引き締まったモノトーンの個性が出て、白文字も余裕で読めます（白文字 on #54504b ＝ 8.0:1）。
また純粋な無彩色より、ほんの少し暖色（greige）に寄せた #e6e2df を使うと、パンフレットの紙の質感ともなじみ「意図した色」に見えます。

⚠ コントラストの基準（色を新規に作るとき）
deep には白文字（成果バッジ・期間ラベル）が載ります。白文字とのコントラスト比が 4.5:1 以上 になる濃さを選んでください。deep2（見出し文字）も tint・soft・白 の各背景に対して 4.5:1 以上を確保します。上の6テーマ（ピンク・オレンジ・グレー・青・緑・黄）はすべてこの基準を満たしています。 WebAIM Contrast Checker で確認できます。

4書体

游ゴシック／ヒラギノ角ゴ系のゴシック体。OS標準フォントのみで、追加読み込みはありません。

連携先名（800・大）

見出しコピー（800・マーカー付き）

リード文（600）多様な人々の視点に立って検討する。

本文（400）案内表示に関する知識や、色覚特性に関する知識など、社会実装を前提とした検討プロセスを学びます。

6ファイル構成

フォルダごと配布・公開できます。サーバーに置けばそのまま動きます。

web/
├─ styles.css　…… デザインシステム本体（共通・編集不要）
├─ style-guide.html　…… このページ（見本と使い方）
├─ jma.html　…… 気象庁の事例ページ（複製のお手本）
└─ images/　…… 各事例の写真

写真は images/ に入れ、HTMLからは images/ファイル名.jpg の相対パスで参照します。1枚目（主写真）と意見交換などの記録写真を .figure で配置してください。
